Le web responsive

Problématique

  • Arrivée des smartphones et tablettes affichant des pages web :
    • sur une multitude d’appareils
    • nombreuses dimensions d’écran
    • 2 formats : portrait et paysage

Solutions

  • Applications natives : Androïd, iOS : nécessite de coder X fois
  • Une version mobile sur un sous-domaine (http://m.site.fr) : problème de "duplicate content"
  • Page web qui se redimensionne selon le dispostif : on ne code qu'une seule fois

Solutions

Responsive : définition

  • "responsive design" , traduire "design adaptatif"
  • Une manière de concevoir un site web pour que son contenu s'adapte automatiquement au terminal utilisé

Exemples

Exemples

  • Site non responsive
  • Un site responsive

Importance

  • Depuis 2015, trafic web sur mobiles > PC
  • Référencement : Google indexe en priorité les sites "mobile-friendly"
  • Inconcevable de créer un site non lisible sur mobile
  • Toujours + d’un site sur 2 non mobile (Yooda, mai 2016)

Techniques à mettre en oeuvre

  • Viewport
  • Points d'arrêt
  • Media queries CSS
  • Système de grille, facultatif mais utile

Le viewport

  • Surface utilisable ("pixels CSS")
  • Différent de la résolution (surface physique)
  • Balise meta (donc dans <head>)

Points d'arrêt = breakpoints

  • Limites de largeur des différents dispositifs

Media queries CSS

  • Spécifications additionnelles CSS (2012)
  • Cible un type de média, une largeur, l'orientation...
  • Bloc d’instructions CSS commençant par @media

Media queries CSS

						
							@media screen and (max-width: 576px) 
							{ 
								h1 
								{ 
									font-size : 20px; 
								} 
							}
  • Se lit : pour les écrans inférieurs à 576px
  • /!\ interprétation : sens max/min
  • /!\ 1 bloc supplémentaire d’accolades

Media queries CSS

  • 2 types d'appels
  • Fichier CSS externe contenant toutes les media queries
  • Séparation selon le dispositif, ciblé par l'attribut "media" de la balise <link>

Media queries CSS

  • Terminaux : screen, print, projection…
  • Sélecteurs : width, orientation, color… + préfixes "min-" et "max-"
  • Règles combinées, opérateurs and, not, only

Système de grille

  • Découpe de la largeur en blocs de largeur égale
  • Lignes + colonnes = cellules
  • Fonctionnalité de base des frameworks CSS (Bootstrap)

Mobile first

  • Philisophie qui préconise d'intégrer en premier les contraintes du mobile.
  • Mobile : avec le moins d’éléments possibles, pour un chargement rapide
  • Puis ajout d’éléments selon la taille d’écran
  • Adopté par Google, Bootstrap...
