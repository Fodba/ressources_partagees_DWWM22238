Javascript

  • Permet de rendre dynamique une page HTML (grâce à des évènements)
  • Langage orienté objet
  • Sensible à la casse : 
    myVar != myvar
  • Norme ECMA : ES6 (= ES2015)

Javascript

  • Côté client (interprété par le navigateur)
  • Librairies/frameworks : Jquery, Angular, VueJS
  • Applications complètes

Appels des scripts

  • Fichier externe : 
    <script src="js/fichier.js"></script>
  • /!\ en fin de page avant la fermeture de 
    </body>
  • /!\ Inutile en HTML 5 : 
    <script type="text/javascript"...

Appels des scripts

  • Dans une balise script directement dans le HTML :
  • Pas d'appel dans une balise 
    <a href="page.html" onclick="alert('Ici !')";>

Syntaxe

  • Mots réservés (if, break, var, try...)
  • Ni espaces ni accents ni tirets, underscore autorisé
  • Langage "typé" : types déclarés (conversions)
  • Concaténation : 
    alert("Bonjour "+myVar);

Types de données

  • Chaîne (string)
  • Nombre (number)
  • Booléen (true/false)
  • Tableaux (array)
  • Objets

2 types particuliers

  • null : absence de données dans une variable ('null' volontairement affecté)
  • undefined : variable existe mais n'a pas de valeur affectée

Manipulation du DOM

  • Modifier la fenêtre (objets Window, Document)
  • Modifier le code HTML et CSS
  • Gérer les cookies

Evènements

  • Action utilisateur déclenche du code Javascript (fonction)
  • Evènements : click, mouseover, keypress, focus, change, submit etc.
  • <a href="page.html" onclick="showmessage();">

Utilisations avancées

  • Ajax (objet XmlHttpRequest) : base de données
  • Frameworks : JQuery, Vue, Angular, React, Meteor etc.
  • Application côté serveur : NodeJS

Outils
